# Токени кольорів

==Семантичні кольори замість raw Tailwind== — основа дизайн-системи.

> [!warning] Заборонено
> `bg-red-500`, `text-green-600` — ESLint блокує commit

## Категорії

| Token | Приклад |
|-------|---------|
| `semantic.*` | success, warning, error |
| `status.*` | connected, validating |
| `atom.*` | problem, solution |

## Імпорт

```typescript
import { semantic, status } from '@/shared/tokens'
semantic.success.bg   // → bg-semantic-success
```

## Маппінг

| ❌ Raw | ✅ Semantic |
|--------|-------------|
| `green-*` | `semantic-success` |
| `red-*` | `semantic-error` |
