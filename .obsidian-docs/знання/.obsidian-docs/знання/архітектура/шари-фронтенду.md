# Шари фронтенду

==Feature-based архітектура== з чіткими залежностями.

## Структура

```
src/
├── app/       # Routing
├── features/  # Domain (18)
├── pages/     # Lazy (18)
└── shared/    # Reusable
```

## Залежності

```
App → Pages → Features → Shared
```

> [!danger] Заборонено
> Features ← Features, Shared ← Features

## Пов'язане

- [[features]] — domain modules
- [[shared-layer]] — reusable код
