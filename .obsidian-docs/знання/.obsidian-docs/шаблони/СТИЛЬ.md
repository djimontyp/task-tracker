# Стиль нотаток

Правила для ==атомарних, легких== нотаток.

## Атомарність

> [!warning] Одна ідея = одна нотатка
> Якщо нотатка > 100 рядків — розбий

## Перелінковка

```markdown
❌ Лінки тільки в кінці
✅ [[лінки]] по тексту
```

## Хайлайти

==Ключове поняття== — 1-2 на секцію.

## Callouts

| Тип | Коли |
|-----|------|
| `> [!warning]` | Заборонено |
| `> [!tip]` | Порада |
| `> [!note]` | Інфо |
