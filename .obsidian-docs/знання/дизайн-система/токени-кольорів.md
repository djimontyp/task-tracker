# Токени кольорів

==Семантичні кольори замість raw Tailwind== — основа [[дизайн-система|дизайн-системи]].

> [!warning] Заборонено
> `bg-red-500`, `text-green-600` — [[eslint|ESLint]] блокує commit

## Категорії

| Token | Коли | Приклад |
|-------|------|---------|
| `semantic.*` | Загальна мета | success, warning, error |
| `status.*` | Стан з'єднання | connected, validating |
| `atom.*` | Типи знань | problem, solution |

## Імпорт

```typescript
import { semantic, status } from '@/shared/tokens'

semantic.success.bg   // → bg-semantic-success
status.connected.text // → text-status-connected
```

> [!tip] Автодоповнення
> TypeScript дає підказки — не треба пам'ятати всі токени

## Маппінг

| ❌ Raw | ✅ Semantic |
|--------|-------------|
| `green-*` | `semantic-success`, `status-connected` |
| `red-*` | `semantic-error`, `destructive` |
| `yellow-*` | `semantic-warning` |
