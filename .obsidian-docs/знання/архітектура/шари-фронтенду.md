# Шари фронтенду

==Feature-based архітектура== з чіткими залежностями.

## Структура

```
src/
├── app/       # Routing, Providers
├── features/  # Domain modules (18)
├── pages/     # Lazy-loaded (18)
└── shared/    # Reusable код
```

## Залежності

```
App → Pages → Features → Shared
```

> [!danger] Заборонено
> - Pages ← Pages
> - Features ← Features
> - Shared ← Features/Pages

## Правила імпортів

| Шар | Імпортує з | Не імпортує з |
|-----|------------|---------------|
| Pages | features, shared | інших pages |
| Features | shared | інших features |
| Shared | shared | features, pages |
