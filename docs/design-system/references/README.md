# Дизайн Референси

Ця папка містить візуальні та концептуальні референси, які надихають дизайн-систему Pulse Radar.

## Призначення

- **Джерело натхнення** для дизайн-рішень та UI патернів
- **Точка відліку** для агентів при оновленні концепцій
- **НЕ для прямої реалізації** — для компонентів використовуй Storybook

## Пріоритет

```
Реалізація:  Storybook (http://localhost:6006)
Концепції:   Ця папка (references/)
```

## Структура

```
references/
├── screenshots/     # Скріншоти UI, візуальне натхнення
├── concepts/        # Текстові описи, mood boards, style notes
├── articles/        # Збережені статті, дизайн-системи інших
└── BOOKMARKS.md     # Посилання на сайти, Dribbble, тощо
```

## Як використовувати

### Додавання референсів

1. **Скріншоти**: Кидай PNG/JPG файли в `screenshots/`
   - Використовуй описові назви: `sidebar-collapsed-notion.png`, `card-hover-linear.png`

2. **Концепції**: Створюй `.md` файли в `concepts/`
   - Опиши що подобається і чому
   - Приклад: `minimal-dashboard-style.md`

3. **Посилання**: Додавай в `BOOKMARKS.md`
   - Категоризуй за типом (dashboards, cards, navigation, тощо)

### Для агентів

При оновленні дизайн-концепцій або створенні нових компонентів:

1. **Спочатку перевір референси** — зрозумій бажану естетику
2. **Шукай патерни** — що спільного між референсами?
3. **Реалізуй в Storybook** — референси інформують, Storybook реалізує
4. **Документуй рішення** — посилайся на натхняючий референс

### Конвенція назв

```
{елемент}-{стиль}-{джерело}.{ext}

Приклади:
- sidebar-minimal-linear.png
- button-hover-stripe.png
- dashboard-dark-notion.png
```

## Категорії референсів

### Візуальний стиль
- Колірні палітри
- Приклади типографіки
- Spacing/layout патерни

### Компоненти
- Cards, buttons, inputs
- Навігаційні патерни
- Візуалізація даних

### Інтеракції
- Hover стани
- Transitions
- Мікро-анімації

### Layouts
- Dashboard layouts
- Mobile responsive
- Empty states

## Поради

- **Якість важливіша за кількість** — курируй, не накопичуй
- **Анотуй** — додавай нотатки про те, що саме надихає
- **Оновлюй регулярно** — видаляй застарілі референси
- **Перехресні посилання** — лінкуй концепції зі скріншотами
